# WebGrenade v4.0.0
Unified utility suite for developers, QA teams, and power users.

![Version](https://img.shields.io/badge/version-4.0.0-orange?style=for-the-badge)
![Manifest](https://img.shields.io/badge/manifest-v3-green?style=for-the-badge)
![License](https://img.shields.io/badge/license-MIT-orange?style=for-the-badge)
![Browsers](https://img.shields.io/badge/browsers-chrome%20%7C%20firefox-blue?style=for-the-badge)

WebGrenade brings practical browser tools into one extension so you do not need to install and maintain many single-purpose add-ons.

## What it includes

### Core
- Unified Dashboard: quick access to every module from one screen.
- Cookie Manager: view, edit, add, and delete cookies on the active domain.
- Popup Blocker: blocks intrusive overlays and aggressive consent modals.
- User-Agent Switcher: test browser/device behavior with custom UA profiles.

### Media and UI
- Volume Booster: amplify tab audio with a 0-300% range.
- Color Studio: eyedropper + palette management for HEX/RGB workflows.
- Global Dark Mode: force dark rendering on unsupported sites.
- Reverse Image Search: launch image lookup from context menu.

### Productivity
- Fake Input Filler: generate realistic test data for forms.
- RSS Reader: follow feeds with auto-discovery support.
- Link Station: shorten URLs and generate QR codes quickly.
- History Cleaner: clear browsing data by domain or time window.

## Installation

### Firefox
- Add-on page: https://addons.mozilla.org/en-US/firefox/addon/webgrenade/
- Manual option: `about:debugging` > temporary add-on

### Chrome / Chromium (Manifest V3)
- Chrome Web Store: https://chromewebstore.google.com/detail/webgrenade/bblgloaglmlfdififfjjflajibogfmnm
- Note: store review can delay update visibility

Gelisticiriciler, QA ekipleri ve ileri duzey kullanicilar icin birlesik tarayici arac seti.

WebGrenade, birden fazla tek amacli eklenti kullanma ihtiyacini azaltmak icin pratik araclari tek bir eklentide toplar.

## Neler var

### Cekirdek
- Birlesik Pano: tum modullere tek ekrandan hizli erisim.
- Cerez Yoneticisi: aktif domaindeki cerezleri goruntuleme, duzenleme, ekleme, silme.
- Popup Engelleyici: rahatsiz edici overlay ve onay modallarini engeller.
- Kullanici Ajani Degistirici: farkli tarayici/cihaz davranisini test eder.

### Medya ve Arayuz
- Ses Guclendirici: sekme sesini %0-300 araliginda artirir.
- Renk Studyosu: damlalik araci ve HEX/RGB palet yonetimi.
- Global Karanlik Mod: desteklemeyen sitelerde zorunlu karanlik gorunum.
- Tersine Gorsel Arama: sag tik menusu ile hizli gorsel arama.

### Uretkenlik
- Sahte Veri Doldurucu: formlar icin gercekci test verisi uretir.
- RSS Okuyucu: otomatik kesif ile feed takibi yapar.
- Link Istasyonu: URL kisaltma ve QR kod uretimi saglar.
- Gecmis Temizleyici: domain veya zaman araligina gore temizleme yapar.
